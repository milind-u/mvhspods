Instructions to Download as a "csv" File

To do this, you first open your spreadsheet with the students and select the "file" option.

Then there should be a "download" option.

After that select the "Comma Separated Values" option and that should download as a csv file.

After you download the file, upload it below.

Click on the "Choose File" button to upload a file:

Instructions to Upload "csv" Pod Groups File Into Google Sheets

First open a new spreadsheet.

Then select the "file" option, and after that select the "import" option.

After that, it should take you to a window which has "My Drive" option selected.

Click on the "Upload" option and select the csv file that you downloaded from the previous set of instructions.
