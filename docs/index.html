Mvhspods

Download students as a *csv* File

To do this, you first open your spreadsheet with the students and click on the the *file*→*download* option. After that select the *Comma Separated Values* option. Then, upload it below and a new *csv* file will be downloaded as `pods.csv` that has the pod groups.

Upload pod groups *csv* into Google Sheets

Open a new spreadsheet. Then select the *file*→*import* option. Now click on the *Upload* option and select the csv file with the pod groups.
